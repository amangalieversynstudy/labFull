# FullStack (Spring Boot)

A Spring Boot REST API with PostgreSQL. Spring Security and JWT have been removed; all endpoints are open. See details in `docs/PROJECT_CONTEXT.md`.

## Quickstart

Requirements:
- Java 21
- PostgreSQL running locally (`localhost:5432/postgres`)

Run (Windows CMD):
```
./mvnw.cmd spring-boot:run
```

Swagger UI: http://localhost:8081/swagger-ui.html

API Docs: http://localhost:8081/api-docs

More details: [docs/PROJECT_CONTEXT.md](docs/PROJECT_CONTEXT.md)
